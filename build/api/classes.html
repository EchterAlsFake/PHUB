class phub.classes.Query(client: Client, url: str, corrector: Callable = None)¶

Represent a generator with list-like properties that contains Video objects.

Example usage:

query = client.search('hello, world!')

# Get a specific video
first_result = query[0]

# Get a range of videos
for video in query[:10]:
    print(video.title)

# List all videos
for video in query:
    print(video.title)
