Error handling#

The following exceptions can be raised by PHUB in case of errors.

exception phub.consts.UserNotFoundError#

Failed to find a PH user account.

exception phub.consts.CounterNotFound#

The video counter wasn’t found in the query, or is badly parsed.

exception phub.consts.ParsingError#

The parser failed to properly resolve the script for this element.
