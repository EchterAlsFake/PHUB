PHUB documentation¶

Introduction¶

PHUB is an API wrapper for Pornhub (PH). It is able to communicate with most used PH features such as video searching, downloading, account connection, etc.

Note

Early dev + i don’t plan to maintain this often so don’t hesitate to submit issues or requests through the project’s github page.
