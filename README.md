# PHUB for Python 3.9

This is a downgraded version of PHUB for python 3.9 and upper. Please note,
that this version is not updated as often as the master branch.

## Installation

```shell
pip install --upgrade git+https://github.com/Egsagon/PHUB.git@py-3.9
```
